    在这里👇，我将教会你如何拯救你的（Windows）电脑💻。

    装系统的U盘

    首先你需要一块可以用来装系统的U盘，（据说）有两种方式：

    1. 使用 Windows 官方工具创建镜像文件到U盘；
    2. 单独下载镜像文件，并使用第三方工具制作 PE U盘。（此类U盘在各网店有售，请酌情购买）

    我的选择是，使用 Windows 官方工具创建镜像文件，存进 PE U盘中。这样既可以选择自己喜欢的（语言版本的）系统，也可以有一块灵活的 PE U盘。

    tips：当你手头没有电脑时，可以去网吧完成这件事——前提是你找到的网吧允许插入U盘。外出前，请查询当地的防疫政策以确保你的核酸证明可以保证你及时回来。

    links:

    1. 参考视频：https://www.bilibili.com/video/BV1DJ411D79y
    2. 下载 Windows 官方工具：https://www.microsoft.com/zh-cn/software-download/windows10
    3. 一个可以下载系统镜像的地方：https://next.itellyou.cn/

    珍惜你的文件与数据！

    幸好，我的 Windows 还没坏掉

    你应该在电脑没有出故障的时候时常为你的文件存档：

    1. 对于视频、音频、各种压缩包等文件，你可以选择自己信任的网盘服务；对于文字性的东西，你大可以存到 GitHub 上，前提是你已经熟悉了它的使用而不至于不小心一下删干净了……（别问我为什么强调这一点）。
    2. 你也可以选择自己购买硬盘进行本地备份。尽管（移动）固态硬盘已经非常成熟，并且这也是我的选择，但是我仍然建议你使用机械硬盘作为存档备份的硬盘仓库——我没有这样做仅仅是因为当时没有人指导我而我现在尚未经济独立。理由是机械硬盘在损坏之后仍然可以进行修复。购买一块大容量的全新硬盘需要几百人民币，但是它给你带来的价值远超这些有形的身外之物。

    完蛋，我的 Windows 已经坏掉

    你发现 Windows 无法正常使用，怀着侥幸心理在网上搜索补救的办法？我将会成为你的英雄，尽管这其实很轻松。

    如果你没有 PE U盘，你应该看装系统的U盘。进入 PE 系统之后，你可以插入外部储存设备（U盘、移动硬盘等）以保存数据。

    快速搭建属于你的 Windows

    哪个版本的 Windows (10)？

    如果你的电脑年代过早，那么这里的所有内容都不适合你。

    最稳的方法是安装 LTSC 版本的 Windows，你可以网上冲浪以了解这是什么东西。但是由于一些三观上的原因，我并没有安装它。我认为最好的选择是你购买笔记本电脑时自带的系统（但是请不要安装家庭单语言版！！！）。什么，你是台式机？那我建议你安装最新的 Windows 10，它相当成熟，并且你可以选择后续升级 Windows 11 if you like。

    至少对于 12 代的 intel CPU，请你无脑冲 Windows 11！

    但是你最好抽个时间看看到底你使用的 CPU 是不是已经被 Windows 11 “优化”过。（我尚不了解详细信息，我所知道的都来自道听途说）

    另外，你还需要在“家庭版”、“专业版”、“企业版”等版本之间选择。通常，个人用户被推荐“专业版”（Pro），因为家庭版缺少很多修复电脑时需要的功能，而企业版增加了很多个人用不到的功能如企业内部的通信。我个人总是选择“专业工作站版”（Pro N），仅仅是为了与众不同。你还应该知道，“教育版”通常会更稳定些，请酌情考虑。

    links:

    1. 一个 Windows 10/11 激活工具：https://cmwtat.cloudmoe.com/cn.html，建议你把它也存在你的 PE U盘里，并在 Windows 安装完毕之后就立刻激活。

    2022 年怎么装软件？

    千万不要使用国产软件商店！尽可能不要从官网下载 setup 程序安装！我建议你使用包管理器 scoop。（此处不提供 scoop 教程）应该指出通过 scoop 安装软件有一些费劲的地方，这些费劲之处来自于它追求一种干净整洁的效果，例如“

    1. 它不会将一些快捷操作加入你的右键菜单中，例如“在 <某某编辑器> 中打开文件夹”，“用 <某某解压软件> 解压缩”等。
    2. 它不会自动设置目标文件的打开方式，在你第一次打开某些文件时，Windows 会让你选择你的打开方式，此时你需要用过文件目录找到 scoop 安装软件的位置直至选中某个可执行文件。

    分享一波我（用scoop）安装了的软件（名称准确）：

    anki, aria2, bandizip, Cascadia-Code, dotnet6-sdk, everything, format-factory, git, googlechrome, LXGWWenKai, LXGWWenKaiMono, mactype-np, miktex, miniconda3, notion, perl, potplayer, powertoys, quicklook, snipaste, steam, sumatrapdf, vcredist-aio, vscode, windows-terminal, yesplaymusic

    如果某个你想要的软件（大部分是国内的）没有在 scoop 中，请在该软件的官方网站下载。

    我也分享一些我从官网下载并安装的软件（名称不准确）：

    Epic，必剪，美图秀秀，tim，微信（scoop 中似乎有），Watt Toolkit，网易有道词典、迅雷、网易云音乐

    腾讯、网易、哔哩哔哩，请赶紧给我打钱！

    2. 安装 MS Office 办公套件：https://otp.landian.vip/zh-cn/，网站内有完整的安装和激活教程

    密码保存

    会有人建议你维护一个物理的密码本，而我使用 Google 账户自动保存浏览器中所需要的密码。你可以选择一种自己喜欢的方式，但请保证你知道这意味着什么，包括好处和坏处。
